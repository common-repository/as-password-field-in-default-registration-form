== Description ==
Plugin put password field in default registraion form of wordpress.


**Usage**

*Basic usage*

Once you have installed and activated the plugin, you will find password field in wordpress registration form.

== Requirements ==
* PHP 5.3 or higher

== Translations ==
* None

== Installation ==
1. Upload the `as-password-field-default-registration` folder to `/wp-content/plugins/`
2. Activate the plugin (AS Password Field In WordPress Default Registration Form) through the 'Plugins' menu in WordPress
3. Check you website's default registration form.

You can leave feature requests in the plugin support forum.
